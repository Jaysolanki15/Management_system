# Snack Manufacturing Management System

A production-ready Next.js, TypeScript, Tailwind, shadcn-style, and Supabase application for managing a family snack manufacturing business.

## Setup

1. Copy `.env.example` to `.env.local` and add your Supabase project URL and anon key.
2. Run the SQL in `supabase/migrations/001_initial_schema.sql` from the Supabase SQL editor.
3. Install dependencies with `npm install`.
4. Start the app with `npm run dev`.

## Modules

- Dashboard and analytics
- Product management
- Shops and customers
- Expenses
- Daily production
- Supabase Auth with RLS
